# Marine-Eco-Vis: Oceanic Movement Analysis

本项目基于达尔豪斯大学（Dalhousie University）相关研究思路，利用 Python 对复杂的海洋生物轨迹数据进行清洗、脱敏及交互式可视化分析。

Interactive visual analytics for marine ecosystem dynamics based on VMSP pattern mining and trajectory data. (基于 VMSP 模式挖掘和轨迹数据的海洋生态交互式可视化分析)

## 🚀 核心功能
- **时空分析**: 基于 GeoMap 展示不同物种（如鲨鱼、鱼类）的全球迁移路径。
- **环境关联**: 挖掘物种活动深度与环境因子的分布规律。
- **模式提取**: 展示年度观测动态，支持海洋保护决策决策。
